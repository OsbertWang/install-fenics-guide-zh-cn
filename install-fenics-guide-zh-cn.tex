\begin{abstract}

  本人 Ubuntu 使用经验并不丰富,
  因此很可能会有记录错误的地方,
  希望可以得到行家里手的批评指正.

  其后续版本 FEniCS-X 还未稳定.
  本文将在 FEniCS-X 发布稳定版本后再更新相关内容.
\end{abstract}
